公司简介

首页/关于我们/公司简介

有机蔬菜，是指在蔬菜生产过程中严格按照有机生产规程，禁止使用任何化学合成的农药、化肥、生长调节剂等化学物质，以及基因工程生物及其产物，而是遵循自然规律和生态学原理，采取一系列可持续发展的农业技术，协调种植平衡，维持农业生态系统持续稳定，且经过有机食品认证机构鉴定认证，并颁发有机食品证书的蔬菜产品。

“有机坊”总占地35亩，有年培育有机蔬菜1万吨生产线一条，年加工1000吨有机菜系列产品生产线三条，均达到国家食品卫生安全检验以及HACCP标准的现代化葛根精深加工标准。总投资2500万元，实现年产值3000万元，提供就业岗位100人以上。

在有机菜系列产品的基础上又相继研发了“精品小黄瓜”“大茄子”“精装圣女果”“高淀粉马铃薯”等一系列养生类产品。2019年“有机坊”被列为重点招商项目及有机菜产业龙头企业，已成为当地农业产业机构调整的重点项目；在政府大力支持下，“有机坊”已成为当地特产的核心产品。20019年至2019年连续4月在全国农业博览会上取得金牌产品的称号；并成为2019-2019年度陕西电视台3.15晚会《都市条形码》、《民生关注》等栏目重点推荐消费者放心食品品牌。

公司本着 “先让自己放心，再使消费者放心”的原则，坚持“社会效益、经济效益、生态效益”的宗旨。以“出优质产品、创知名品牌、报效社会、壮大自身”的目标。让“御春农”产品走进千千万万消费者心中，让食疗代替药疗的知识深入千家万户，让“御春农”产品带给人民健康幸福、美丽快乐。我们真诚期盼与社会各界有识之士携手共进、共同开创健康产业的美好未来。
