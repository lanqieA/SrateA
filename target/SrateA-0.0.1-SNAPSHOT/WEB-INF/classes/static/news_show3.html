  • 宏鸿农产品315宣言 | 始终坚守食品安全为根本

  • 2019.4.25 10:10:58来源:公司内部点击：0点评：0

  • 中国消费者协会将今年（2019年）的消费维权主题定为“信用让消费更放心”，其目的在于推动消费者成为信用提升的推动者和监督员，共同建设品质消费生态圈，形成信用提升与社会共享共治新机制。

    近年来，食品安全领域中损害消费者权益的情况也不断在发生，经营者信用缺失的现状情况也不容乐观，在很大程度给消费者的满意度和消费信心带来了一定的冲击，也制约着消费潜力的进一步扩大。 多年以来，作为团餐食材配送领域的专家——有机坊农产品集团一如既往坚守“与您携手，创造优质生活”的企业使命，全面落实从“田间到舌尖”的农产品质量安全问题，为全国消费者配送安全、健康的食材。有机坊集团始终坚守食品安全底线，赢得了消费者的认可和信赖。

    为了打造安全可靠的农产品质量管理体系，确保有机坊配送的食材符合高品质的质量标准，宏鸿打造了完善的农产品溯源系统。该系统能够方便客户全程掌握食材生产到配送的每一个环节，从种植、加工，检验、运输等多个环节，消费者都可以通过扫描二维码了解到各项信息，这不仅可以让消费者们买的放心，更可以吃的安心。 宏鸿农产品集团在近年来先后获得“广东省名牌产品”“广东省诚信企业”、 “广东省守合同重信用企业”、“ AAA级企业信用等级证书” “质量服务AAA级诚信单位”以及“售后服务评价‘五星级’认证企业 等荣誉称号。这一项项闪耀荣誉的背后，是宏鸿集团多年如一日地对食品安全矢志不渝的追求。

    诚信，有机坊一直在行动；品质，有机坊一直在坚持。食品的终极使命是为健康服务。随着《“健康中国2030”规划纲要》的颁布，健康成为国家战略部署的重要组成部分。有机坊农产品集团将持续以匠心精神保证食品品质与安全，为振兴农产品配送行业和国人健康而努力。
